# Assignment 05A: Decision Tree - Diabetes

## Files
- `05_dt_diabetes.py`
- `diabetes.csv`
- `shared_drive_05_diabetes_decision_tree.py`
- `drive_diabetes.csv`

## Packages Required
```bash
python -m pip install pandas numpy matplotlib scikit-learn
```

Or install the common project requirements from the project root:
```bash
python -m pip install -r requirements.txt
```

## How to Run
From the project root:
```bash
python 05A_Decision_Tree_Diabetes/05_dt_diabetes.py
```

If using VS Code, select the interpreter from your local `.venv` folder.

## Shared Drive Version
The shared Drive version is `shared_drive_05_diabetes_decision_tree.py`.

Run it from the project root:
```bash
python 05A_Decision_Tree_Diabetes/shared_drive_05_diabetes_decision_tree.py
```
